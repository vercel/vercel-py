Blob SDK Examples (Python)

Prereqs:
- Python 3.10+
- Env `BLOB_READ_WRITE_TOKEN` set (server-side examples)
- Install deps: `pip install httpx`

Examples:
- server_basics.py: put, list, head, copy, delete, folder
- multipart_examples.py: uncontrolled and manual multipart

Run:
  python server_basics.py
  python multipart_examples.py
